DOCUMENTACIÓN DEL DESARROLLADOR

Referencia técnica para la integración de aplicaciones con la matriz neuronal AETHER.

warning

Alpha v0.9.8

El SDK requiere una llave biométrica de nivel 4 para el acceso a la red neuronal pública. Contacte con el Comando Central para autorizaciones.

Quick Start

Instala el cliente oficial mediante el gestor de paquetes:

npm install @aether-core/neural-link

Implementación

Inicializa el enlace con latencia cero: 

const client = new AetherClient({
  mode: 'quantum',
  latency: 0.01
});

await client.link();
